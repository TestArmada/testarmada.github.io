Download Options

Get TestArmada just the way you want it. We officially distribute via npm, bower, and static builds on this site.

Download Information v2.4.2

You can download TestArmada either with npm or bower or as direct file download.

Install with bower

bower install marionette

Install with npm

npm install backbone.marionette

Static Builds

Pre-Packaged

These archives contain all of the files you need to get started with TestArmada, including Backbone, jQuery and all other prerequisites.

These packages include the standard UMD version of both the non-minified and minified versions of TestArmada.

*nix (.tar.gz)

backbone.marionette.tar.gz

Windows (.zip)

backbone.marionette.zip

Bundled (UMD)

Download the bundled backbone.marionette.js file with the Backbone.Wreqr and Backbone.BabySitter prerequisites built in.

  • backbone.marionette.js
  • backbone.marionette.min.js

Core

Download the core backbone.marionette.js file, without Backbone.BabySitter or Backbone.Wreqr.

These pre-requisites are still required for TestArmada to run, but this allows you to download them separately and update them independently.

Standard .js (UMD)

  • backbone.marionette.js
  • backbone.marionette.min.js
